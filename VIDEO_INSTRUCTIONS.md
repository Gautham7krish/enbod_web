# Adding Your Background Video

To add your video file to the website:

1.  Rename your video file to `bg-video.mp4`.
2.  Place the file in the `public` folder of the project.
    - Path: `c:\Users\gauth\OneDrive\Desktop\new folser\public\bg-video.mp4`
3.  The website will automatically pick it up and use it as the background for the hero section.

If you have a different file extension (e.g., `.mov` or `.webm`), you can update the `src` attribute in `src/components/VideoBackground.jsx`.
